# はじめに

これはサンプルのMDX記事です。このブログ機能では、`posts`フォルダに`.mdx`ファイルを追加するだけで、新しい記事を公開できます。

## 機能

- Markdown形式で記事を書けます
- MDXを使用することで、Reactコンポーネントも埋め込めます
- フロントマターでメタデータ（タイトル、日付、説明）を管理できます

## コードブロック

コードブロックも使えます：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("World");
```
